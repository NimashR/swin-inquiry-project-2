Enhancements

This page lists all enhancements made to the website. The links at the beginning of each item redirect to the place where the enhancement in question is integrated.

Additional Elements

Various HTML and CSS elements have been incorporated outside of those taught in the lectures. These include:

  1. An image map on the 'Home' page, which provides links to all other pages on the website. This makes the site more approachable to new visitors by including a large, colourful and easily readable navigation section.
    The image map is created by including an img element with the attribute usemap. This references another HTML element, map, which itself contains several area elements that provide the co-ordinates (coords) for the clickable area on the signs, and the pages to be redirected to when those areas are clicked (href).
  2. Figure tags on the 'Usage' page that provide more description to images. This helps clarify what information they include, and ensures that the user can quickly identify the purpose of each image.
    The figures are created by making the img image elements children of figure elements, then including an accompanying figcaption element to provide the additional information.
  3. Code tags on this page that distinguish code from the other text in the paragraph. This makes the explanation sections for each enhancement much more legible by making it clear the exact spelling and syntax of the code used.
    To create these code snippets, the code tag is used as an inline element inside each block of text.

CSS Bootstrap Integration

The preprocessor Bootstrap has been included on all pages. This provides several benefits, including quick and easy styling of key elements, as well as functionality not present in standard HTML and CSS. The following additions have been made:

  1. An responsive, embedded YouTube video on the 'Home' page that allows for viewing the video on the home page itself, instead of having to use the link to visit YouTube.com (though this is still an option). This provides more convenience, as well as better availability for users with poor Internet connections, since it only requires them to load the video instead of the entirety of YouTube.
    To embed the video, the iframe element is placed as the child of a section element, with the link to the video specified with the src attribute. The Bootstrap classes embed-responsive embed-responsive-16by9 and embed-responsive-item then provide the video with proper scaling as the size of the web browser window changes.
  2. The submit and reset buttons on the 'Quiz' page have been styled using Bootstrap. This makes them easier to see, and clarifies their purpose using widely understood conventions of green as success and red as failure.
    To do this, the submit button has had the btn btn-success class applied to it, while the reset button has been given the btn btn-danger class.
